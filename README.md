# StudyPharm
StudyPharm is a mobile application designed to assist pharmacy students in studying and memorizing the top 200 pharmaceutical drugs commonly used and prescribed. The app provides detailed information about each drug, including brand name, generic name, drug class, indication, and schedule. Additionally, users can take personalized notes related to each drug to aid in their study process. It utilizes the Flutter framework and Firebase database.
